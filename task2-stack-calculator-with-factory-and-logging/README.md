# Itroduction

**jcalc** - is a stack calculator written in java. The name jcalc stands for **j**ava **calc**ulator. 

jcalc supports such commands like:
- `help` - print list of commands and their descriptions
- `push/pop` - push/pop value on/from stack 
- `+, -, *, /, SQRT` - arithmetic operations
- `print` - print top element of a stack
- `define` - define a parameter
- ` ` - secret command ???

# Example
Input:
```
DEFINE a 4
PUSH a
SQRT
PRINT
```
Output:
```
2
```
