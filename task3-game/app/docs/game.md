# О чём вообще игра?

Это простой tower-defence. Есть база, есть враги, которые хотят уничтожить базу. 
И есть тропа, по которой они идут. Вдоль тропы можно ставить башни, которые будут стрелять по врагам.

# Реализация

## Overview

В лабе указано, что надо использовать паттерн Model-View-Controller, а значит
графическое представление работает независимо от логики игры.

### Model (логика)

Разбиваем поле на столько клеток, сколько требуется для уровня. Нарпимер, 10x10. Будет 2 типа объектов

1. **Статические** - их можно будет ставить только целиком на всю клетку. Далее - *static*
Можно у каждой клетки хранить массив объектов на ней?

2. **Свободно перемещающиеся** - эти объекты могут иметь произвольные координаты. Далее - *dynamic*

#### Объекты

1. Трава *static*. Блоки 1х1.
2. Тропинка *static* Блоки 1х1.
3. Враги *dynamic* - могут перемещаться только по тропинке
4. Башни *statc* - не могут стоять на тропинке. 
5. Здание *static* - их хотят разрушать враги
6. Снаряды *dynamic* - их выпускают башни. Наносят дамаг врагам.
